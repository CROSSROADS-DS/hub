# Hub Source

This directory contains the canonical development source for the public
CROSSROADS Hub.

The Hub will eventually include:

- the public project website;
- the module catalog;
- links to public repositories;
- direct links to JupyterLite notebooks;
- project documentation;
- a link to the Forum;
- the JupyterLite build configuration;
- the GitHub Actions deployment workflow.

During publication, approved files from this directory will be copied to
the public `crossroads-ds/hub` repository.
